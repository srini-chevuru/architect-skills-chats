## What happens when you save multiple times

| Mode | Filename | Behaviour |
|---|---|---|
| **Default** (no flag) | `2026-06-19-my-topic.md` | **Overwrites** the same file — last save wins |
| **`--append`** | `2026-06-19-my-topic.md` | **Appends** to the existing file below a `--- appended: 2026-06-19 HH:MM ---` separator |
| **`--unique`** | `2026-06-19-21-31-my-topic.md` | Always creates a **new timestamped file** (date + hour + minute) |

**Use `--append`** when you're iterating on the same topic across sessions and want one running document per topic per day.

**Use `--unique`** when you want a clean archive of every individual session.

```zsh
# Grow a single running doc for today
save-chat-to-space chat.md --space architect-skills --title "roster-design" --append

# Always create a fresh snapshot
save-chat-to-space chat.md --space architect-skills --title "roster-design" --unique
